前言

近日受国家安全政策影响，Google相关服务基本处于全面被墙状态，虽然限制互联网自由本身是不合理，但以发展的眼光去认真思考下GFW的作用或许可以成为一把双刃剑，本文主要结合实际分享简单好用的方法，如果有更优雅的解决方案我也会持续更新，也希望大家积极反馈，ok, let it go

免费方案

各个平台的解决方案都不唯一，请认真阅读原文中的说明部分

Buckyball

轻松访问Twitter，Facebook，Youtube的浏览器

  • 【首选】Windows翻墙方案

https://code.google.com/p/buckyball/

GoAgent+

一个支持 GoAgent 和 Shadowsocks 客户端的代理软件

  • 同时支持 GoAgent 和 Shadowsocks
  • 可以自动获取免费公开服务器
  • 完善的自定义代理设置

https://goagentplus.com/

Goagent

a gae proxy

https://github.com/goagent/goagent

Goagent_Assistant

Goagent自动下载、安装、更新、配置工具

https://bintray.com/zane/Goagent/Goagent_Assistant/

Shadowsocks

A fast tunnel proxy that help you get through firewalls

https://github.com/clowwindy/shadowsocks-csharp

fqrouter

手机变身翻墙路由器，从此科学上网无压力。

  • 【首选】Android翻墙方案

http://fqrouter.com/

GoAgentX

GoAgentX is a GUI application on OS X.

  • 【首选】OS X翻墙方案

https://github.com/ohdarling/GoAgentX

iOS

shadowsocks client for OSX and non-jailbroken iOS

  • 【首选】iOS翻墙方案

https://github.com/shadowsocks/shadowsocks-iOS

Windows Phone

  • 暂无直接方案，建议使用类似Shadowsocks的Share over LAN功能设置局域网代理

付费方案

请根据自身实际需求，在认真阅读网站说明后做出合理的选择

Shadowsocks商业版

A lightweight tunnel proxy which can help you get through firewalls

  • 【首选】由Showfom和Shadowsocks的发明人clowwindy一起运营
  • 我已经在第一时间入坑高级版至2018年
  • 信任.Trust，以下是我唯一的推荐链接

https://portal.shadowsocks.com/aff.php?aff=043

VPNso

稳定靠谱，良好的口碑

https://killwall.com/

曲径

使用曲径，即可在中国的各个角落，尽情地欣赏来自世界各地的海量音乐。无论是电脑还是手机，也不管是Wi-Fi抑或4G，曲径都能始终伴随你左右。更棒的是，借助我们先进的分流技术，你使用其它互联网服务丝毫不受影响。

https://getqujing.com/
